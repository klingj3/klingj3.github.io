Skills and Projects

Automatic Microfossil Concentrate Sorter

In paleontology, microfossil concentrate is a mixture of microfossils and gravel, with each fossil being about 2 mm in size. Microfossils can potentially serve as a source of big data for paleontologists about the locations and environments from which they were collected, however the sorting and categorizing these fossils is a very time consuming process. I designed the software for a machine to automatically sort these fossils using Python and common visual and data libraries such as OpenCV, Numpy, and scikit-learn.

Reddit Political Sentiment Explorer

One emerging application of Natural Language Processing is the tagging of political speech. This project is a first step towards this goal, by using a combination of LSTM-backed sentiment classification with entity linking for determining the affiliation of individual comments. Presently, this tool is applied in a web application for exploring the differing ideological affiliations expressed in news article discussions across Reddit, though the upcoming goal for this project is to generate a robust corpus for political sentiment classification, as the no good options currently exist.
This is an ongoing project.
