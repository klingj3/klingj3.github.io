Social Media Political Sentiment Analyzer

One emerging application of Natural Language Processing is the tagging of political speech. This project is a first step towards this goal, by using a combination of LSTM-backed sentiment classification with entity linking for determining the affiliation of individual comments. Presently, this tool is applied in a web application for exploring the differing ideological affiliations expressed in news article discussions across Reddit, though the upcoming goal for this project is to generate a robust corpus for political sentiment classification, as the no good options currently exist.
